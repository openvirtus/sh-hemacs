## HEMACS

    Usage: hemacs ...
    
    This script helps configuring and managing the GNU/Emacs text editor.
    
    -v         : Print environment variables.
    -P PACKAGE : Install package.
    -S         : Install basic setup.
                 - Adds all directories in `~/.emacs.d/packages`.
                 - Adds ~/.emacs.d/themes to theme path.
                 - Loads all files in `~/.emacs.d/profile.d/*.el`.
    -F         : Install `flycheck`, `yasnippet`, `company`, `sr-sidebar` and `yafolding`.
                 - Put a link in ~/.yasnippet to your directory.
    
    ## Manage chunks in [~/.emacs.d/init.el] ##########################
    
    -A NAME      < CODE.el : Add code named `NAME` into `init.el`.
    -I NAME      < CODE.el : Insert code.
    -B NAME=TEXT < CODE.el : Add button to run standard input code.
    -D NAME                : Delete code named `NAME` from `init.el`.
    -L                     : List code added to `init.el`
